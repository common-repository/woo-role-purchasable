== Description ==

Makes Woocommerce products purchasable according to user's role.
Sometimes you do not want guests or certain users can purchase products, simply show them, but they are not purchasable.

== Installation ==

1. Upload or extract the `woorolepurchasable` folder to your site's `/wp-content/plugins/` directory. Or you could use the *Add new* option found in the *Plugins* menu in WordPress.  
2. Enable the plugin from the *Plugins* menu in WordPress.
3. A new *Role Purchasable* submenu will appear in Woocommerce menu, this is where you manage the settings.
4. Go to Woocommerce > Role Purchasable and select the roles or guests.
